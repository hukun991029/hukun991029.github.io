一.初识 Vue

1.1 Vue 初使用

  1. v-once:只调用一次不能第二次调用
  2. v-html：将 html 标签解析
  3. v-test：<p v-text='message'>,hhh</p> 会将后面的文本覆盖(不灵活 较少使用)
  4. v-pre:<p v-pre>{{message}}</p>将标签内容原封不动的显示
  5. v-cloak:(cloak 斗篷的含义)可以解决插值空白的问题
  6. v-bind:动态绑定
  7. v-for: v-for="(item, index) in items" :key="index"

1.2 computed 计算属性

  1. 计算属性一般不写 set 方法
  1. 简写
  1. 计算属性回家乡缓存，多次使用时计算属性只会调用一次
  2. 方法不会缓存，计算属性有缓存，可以节省性能

1.4 const

  1. 一旦给 cosnst 修饰的标识符被赋值后，不能修改
  2. 使用前必须赋值
  3. 变量的含义是指向的对象不能修改，但是可以改变对象内部的属性
